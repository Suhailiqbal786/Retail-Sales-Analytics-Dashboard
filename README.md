# 📊 Task 03 – Retail Sales Data Cleaning, Analysis, and Visualization

## 📌 Project Overview

This project focuses on cleaning, analyzing, and visualizing a retail sales dataset using Microsoft Excel, SQL, Python, and Power BI. The objective is to transform raw sales data into meaningful business insights through data cleaning, analysis, and an interactive dashboard.

---

## 🎯 Objective

* Clean and prepare the retail sales dataset.
* Analyze the data using SQL and Python.
* Build an interactive Power BI dashboard.
* Generate business insights to support decision-making.

---

## 🛠️ Tools & Technologies

* Microsoft Excel
* SQL
* Python (CSV & Pandas)
* Power BI Desktop

## 📊 Dashboard Features

* KPI Cards

  * Total Revenue
  * Total Orders
  * Total Customers
  * Average Rating
  * Average Discount

* Revenue by Category

* Revenue by City

* Revenue by Payment Method

* Orders by Status

* Monthly Revenue Trend

* Interactive Slicers

  * Category
  * Status
  * City

---

## 📈 Key Insights

* Furniture generated the highest revenue.
* Surat was the top-performing city by revenue.
* Delivered orders had the highest number of orders.
* Interactive filters enable dynamic dashboard exploration.
